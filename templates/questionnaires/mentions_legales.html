Mentions légales et confidentialité

Cookies et données personnelles

Quels cookies utilisons-nous ?

Ce site utilise uniquement des cookies strictement nécessaires à son fonctionnement :

  • Cookie de session Django (sessionid) : permet de maintenir votre connexion et de sauvegarder votre progression dans les questionnaires
  • Cookie CSRF : protège le site contre les attaques de type Cross-Site Request Forgery

Durée de conservation : 4 heures

Aucun cookie publicitaire ou de tracking n'est utilisé sur ce site.

Comment refuser les cookies ?

Vous pouvez configurer votre navigateur pour refuser les cookies. Cependant, cela empêchera le fonctionnement normal du site.

Données personnelles collectées

Dans le cadre de l'utilisation de ce site, nous collectons :

  • Pour les clients : numéro SIREN, nom de l'entreprise, réponses aux questionnaires
  • Pour les collaborateurs : adresse email, nom, prénom

Finalité du traitement

Les données sont collectées dans le but exclusif d'établir un diagnostic de votre situation concernant la facturation électronique 2026 et de vous proposer un accompagnement adapté.

Durée de conservation

Les données sont conservées pendant 1 an à compter de leur collecte.
